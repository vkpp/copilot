# Task Manager

Small accessible task manager web app.

Features:
- Add tasks
- Mark tasks complete / undo
- Delete tasks
- Clear completed
- Persistence via `localStorage`

Usage:
1. Open `index.html` in your browser.
2. Type a task and press Enter or click Add.
3. Use the Done/Undo and Delete buttons to manage tasks.
